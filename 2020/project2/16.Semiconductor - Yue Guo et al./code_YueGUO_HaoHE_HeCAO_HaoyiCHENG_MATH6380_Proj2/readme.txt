This package contains the source code of MATH6380P Project II: Image Classification ofSemiconductors.

EDA.ipynb is the code for Exploration Data Analysis.
model-res50-baseline.ipynb is the code of baseline model (ResNet50) and all the refinement tricks.
model-effb7 is the code of EfficientNet-b7 (which achieves best result).
Ensemble.ipynb is the code for ensemble models.
